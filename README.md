## 레이드 타이머

- 인게임에서 레이드 시간을 확인하기 위해서는 해당 지역으로 이동해서 확인해야하는 단점이 있어 만들게 된 레이드 타이머
- 해당 지역에 있는 유저들이 직접 제보하는 시스템
- 상단 바에서 알람 버튼 클릭하여 브라우저 알람을 받을 수 있고, 원하는 알람 시간을 설정할 수 있다.
